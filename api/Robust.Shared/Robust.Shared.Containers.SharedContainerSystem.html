IsInSameOrTransparentContainer(EntityUid, EntityUid, BaseContainer, BaseContainer, Boolean)

Check whether a given entity can see another entity despite whatever containers they may be in.

Declaration
public bool IsInSameOrTransparentContainer(EntityUid user, EntityUid other, BaseContainer userContainer = null, BaseContainer otherContainer = null, bool userSeeInsideSelf = false)
Parameters
Type Name Description
EntityUid user
EntityUid other
BaseContainer userContainer
BaseContainer otherContainer
System.Boolean userSeeInsideSelf
Returns
Type Description
System.Boolean
Remarks

This is effectively a variant of IsInSameOrParentContainer(EntityUid, EntityUid) that also checks whether the containers are transparent. Additionally, an entity can "see" the entity that contains it, but unless otherwise specified the containing entity cannot see into itself. For example, a human in a locker can see the locker and other items in that locker, but the human cannot see their own organs. Note that this means that the two entity arguments are NOT interchangeable.
