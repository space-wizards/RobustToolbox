LastProcessedTick

This is functionally the clients "current-tick" before prediction, and represents the target value for LastRealTick. This value should increment by at least one every tick. It may increase by more than that if we apply several server states within a single tick.

LastRealTick

The last real non-extrapolated server state that was applied. Without networking issues, this tick should always correspond to LastRealTick, however if there is a missing states or the buffer has run out, this value may be smaller..
