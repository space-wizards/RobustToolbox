Class RSI.State

Represents a single icon state inside an RSI.

Inheritance
System.Object
RSI.State
Implements
IRsiStateLike
IDirectionalTextureProvider
Namespace: Robust.Client.Graphics
Assembly: Robust.Client.dll
Syntax
public sealed class State : object, IRsiStateLike, IDirectionalTextureProvider
Remarks

While the RSI spec allows different animation timing for directions in the same frame, RSIs are folded into a single set of animation timings when loaded. This is to simplify animation playback code in-engine.
