PartialStateReset(GameState, Boolean, Boolean, Boolean)

This will perform some setup in order to reset the game to an earlier state. To fully reset the state ApplyGameState() still needs to be called separately.

Declaration
void PartialStateReset(GameState state, bool resetAllEntities, bool deleteClientEntities = false, bool deleteClientChildren = true)
Parameters
Type Name Description
GameState state

The state to reset to.
System.Boolean resetAllEntities

Whether or not to reset LastStateApplied for all entities, or only those that have been modified some after the states ToSequence. This effectively determines whether we should do a full-state reset, or only reset recently modified entities.
System.Boolean deleteClientEntities

Whether to delete all client-side entities (which are never part of the networked game state).
System.Boolean deleteClientChildren

Whether to delete client-side entities that are parented to networked that are about to be deleted during the partial reset. E.g., if this is true, then a client-side muzzle flash effect entity that is parented to a networked gun entity will get deleted if that gun is about to be deleted. If false, the entity will simply be detached to nullspace. This option has no effect if is true.
Remarks

This function will delete any networked entities that are not present in the given game state. Any child entities that are in the state will simply be sent to null-space. This will also reset LastStateApplied to zero, so that ApplyGameState() will actually apply the state.
