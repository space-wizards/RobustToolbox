Class RobustIntegrationTest.IntegrationInstance

Provides control over a running instance of the client or server.

Inheritance
System.Object
RobustIntegrationTest.IntegrationInstance

Namespace: Robust.UnitTesting
Assembly: Robust.UnitTesting.dll
Syntax
public abstract class IntegrationInstance : IDisposable
Remarks

The instance executes in another thread. As such, sending commands to it purely queues them to be ran asynchronously. To ensure that the instance is idle, i.e. not executing code and finished all queued commands, you can use WaitIdleAsync(Boolean, CancellationToken). This method must be used before trying to access any state like ResolveDependency<T>(), to prevent race conditions.
