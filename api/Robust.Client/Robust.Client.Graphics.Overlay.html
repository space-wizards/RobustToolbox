Class Overlay

An overlay is used for fullscreen drawing in the game. This can range from drawing parallax to a full screen shader.

ZIndex

Overlays on the same OverlaySpace will be drawn from lowest ZIndex to highest ZIndex. As an example, ZIndex -1 will be drawn before ZIndex 2. This value is 0 by default. Overlays with same ZIndex will be drawn in an random order.

BeforeDraw(OverlayDrawArgs)

This function gets called prior to the overlay being drawn. If this function returns false, the overlay will not get drawn to this view-port. Useful for avoiding unnecessary screen-texture fetching or frame buffer clearing.
