Redial(String, Nullable<String>)

Try to cause the launcher to either reconnect to the same server or connect to a new server. The engine will shutdown on success. Will throw an exception if contacting the launcher failed (success indicates it is now the launcher's responsibility). To redial the same server, retrieve the server's address from LaunchState.Ss14Address.
