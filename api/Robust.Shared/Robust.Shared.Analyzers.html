AutoGenerateComponentStateAttribute

When a component is marked with this attribute, any members it has marked with AutoNetworkedFieldAttribute will automatically be replicated using component states to clients. Systems which need to have more intelligent component state replication beyond just directly setting variables should not use this attribute.
