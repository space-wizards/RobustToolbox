Namespace Robust.Shared.Toolshed

Classes

CommandArgumentAttribute

Marks an argument in a function as being an argument of a ToolshedCommand. This will make it so the argument will get parsed.

CommandImplementationAttribute

Marks a function in a ToolshedCommand as being an implementation of that command, so that Toolshed will use it's signature for parsing/etc.

CommandInvertedAttribute

Marks an argument in a function as specifying whether or not this call to a ToolshedCommand is inverted.

CommandInvocationContextAttribute

Marks an argument in a function as being where the invocation context should be provided in a ToolshedCommand.

PipedArgumentAttribute

Marks an argument in a function in a ToolshedCommand as being the "piped" argument, the return value of the prior command in the chain.

TakesPipedTypeAsGenericAttribute

Marks a command implementation as taking the type of the previous command in sequence as a generic argument.

ToolshedCommand

This class is used for implementing new commands in Toolshed.

ToolshedCommandAttribute

Used to mark a class so that ToolshedManager automatically discovers and registers it.

ToolshedEnvironment

ToolshedManager

The overarching controller for Toolshed, providing invocation, reflection, commands, parsing, and other tools used by the language. External documentation has a more in-depth look.
