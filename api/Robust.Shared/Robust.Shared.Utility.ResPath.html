Extension

Returns the file extension of ResPath, if any as string. Returns "" if there is no file extension. (Hidden) Files starting with period (".") are counted as files with no extension. The extension returned does NOT include a period.

    Assert.AreEqual("png", picPath.Extension);

Filename

Returns the file name (folders are files) for given path, or "." if path is empty. If last segment divided of a path (e.g. /foo/bar/baz.png) divided by separator (e.g /) is considered a filename (e.g. baz.png).

    // folder

    // empty

FilenameWithoutExtension

Returns the file name part of a ResPath, as string. In essence reverse of Extension. If last segment divided of a path (e.g. /foo/bar/baz.png) divided by separator (e.g /) is considered a filename (e.g. baz.png). In that segment part before period is considered filename (e.g. baz, unless file start with period, then whole segment is filename without extension.

Declaration
public string FilenameWithoutExtension { get; }
Property Value
Type Description
System.String
Examples
    // file with normal extension
    var picPath = new ResPath("/a/b/foo.png");
    Assert.AreEqual("foo", picPath.FilenameWithoutExtension());
    // hidden file starting with `.`
    var gitignore = new ResPath("/a/b/.gitignore");
    Assert.AreEqual(".gitignore", gitignore.FilenameWithoutExtension());
